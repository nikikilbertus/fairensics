Fairensics is a python library to discover and mitigate biases in machine learning models and datasets.
The best location to learn about fairensics are the jupyter notebooks_.

Fairensics is based on AIF360_ and provides compatible versions of the fairness methods found here_.

.. _notebooks: https://github.com/nikikilbertus/fairensics/tree/master/examples
.. _AIF360: https://aif360.mybluemix.net/
.. _here: https://github.com/mbilalzafar/fair-classification
